# Bankomaten 2.0


## Inlämningsuppgift
Kopiera er bankomat uppgift och ändra till att spara med en lista istället för en array. 
Vid avslut ska listan serializeras och sparas i en fil, så att ni automatiskt kan ladda listan när programmet startas.

Frivillig extra del:
Implementera funktionallitet för att lägga till och ta bort konton så valfritt antal kan finnas.
